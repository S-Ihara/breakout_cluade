# ブロック崩し (Breakout Game)

シンプルで楽しいブロック崩しゲームです。ブラウザで動作します。

## 🎮 遊び方

[GitHub Pagesでプレイ](https://s-ihara.github.io/breakout_cluade/)

## 📋 ゲーム説明

- パドルを動かしてボールを跳ね返し、すべてのブロックを壊すゲームです
- ボールを落とすと残機が減ります
- すべてのブロックを壊すとクリアです

## 🎯 操作方法

- **左右移動**: 左右矢印キー または A/D キー
- **ゲーム開始**: スペースキー またはスタートボタン

## 🎨 特徴

- レスポンシブデザイン
- カラフルなブロック
- スコアシステム
- 残機システム
- スムーズなアニメーション

## 🚀 ローカルで実行

1. リポジトリをクローン:
```bash
git clone https://github.com/S-Ihara/breakout_cluade.git
cd breakout_cluade
```

2. `index.html` をブラウザで開くだけで遊べます
